### 設計画面
駒数は一旦将棋準拠の20。持ちポイントは
そのうち、玉・王は駒の能力を変えることができない。

駒の配置は、将棋とまったく同じで変えることはできない。

最初の持ちポイントは各ポイントで
9 + (4x4 + 6 + 5 + 4x2) x2 = 79(point)

プレイヤー1が王、プレイヤー2が玉
駒は
前、後、横、ななめ、それぞれ1マスに移動する能力に1ぽいんと。たとえば金は前、横2箇所、ななめ2箇所、うしろ1っ箇所の合計6箇所に移動できるため、6ポイント仕様
香車、角のようなななめに移動する能力に4ポイント
桂馬のような、ニマス離れた場所に移動する能力に2ポイント

駒がなると
固定で+2ポイントまで能力を買う

### プレイ画面
ターン毎に画面が反転するようにする。
画面切り替えはディゾルブ
プレイヤー2のときは背景強調色にする

### 勝利条件
詰むようにするか、玉・王をとると勝つようにするか
いったんとるとかつように

# 
このゲームは、作成フェーズとプレイフェーズという、2つの時間で構成されている。

## プレイ（簡略ルール）
- 9×9、持ち駒あり（打てます）。
- 玉/王を取ったら勝ち（詰み判定なし）。
- 二歩などの細かい禁則は未実装。
- 成り：成れる駒は、成り時に追加能力を最大 `+2` まで購入できます（UI簡略のためスライドは購入対象外）。
